## üèó Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation & Development
```bash
# 1. Install dependencies
npm install

# 2. Start the development server
npm run dev

# 3. (Optional) Run Genkit/AI development mode
npm run genkit:dev
```

### Production
```bash
# Build for production
npm run build

# Start production server
npm start
```
